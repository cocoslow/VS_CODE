# 안녕하세요!

이 파일은 요청하신 마크다운(Markdown) 예제 파일입니다.

## 마크다운 문법 안내

마크다운은 텍스트에 기호를 사용하여 서식을 지정할 수 있는 가벼운 언어입니다. 자주 쓰이는 문법들은 아래와 같습니다.

### 1. 텍스트 강조

- **굵은 글씨** (Bold)
- *기울인 글씨* (Italic)
- ~~취소선~~ (Strikethrough)

### 2. 목록 만들기

**순서가 없는 목록:**

* 첫 번째 항목
* 두 번째 항목
  * 하위 항목

**순서가 있는 목록:**

1. 첫 번째
2. 두 번째
3. 세 번째

### 3. 코드 작성

간단한 코드나 텍스트를 강조할 때는 `code` 처럼 백틱(`)을 사용합니다.

여러 줄의 코드는 아래와 같이 작성합니다:

```javascript
function hello() {
  console.log("Hello, Markdown!");
}
```
