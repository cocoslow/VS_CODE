삼각비 — 2개 알면 나머지 계산
알고 있는 값 구하는 공식
R + θ X = Cx + R×cos(θ)
Y = Cy + R×sin(θ)
X + Y dx = X−Cx, dy = Y−Cy
R = √(dx²+dy²)
θ = atan2(dy, dx)
R + X dx = X−Cx
cos(θ) = dx/R
θ = acos(dx/R)
Y = Cy + R×sin(θ)
R + Y dy = Y−Cy
sin(θ) = dy/R
θ = asin(dy/R)
X = Cx + R×cos(θ)
θ + X dx = X−Cx
R = dx / cos(θ)
Y = Cy + R×sin(θ)
θ + Y dy = Y−Cy
R = dy / sin(θ)
X = Cx + R×cos(θ)
항등식 sin²θ + cos²θ = 1
tan(θ) = sin(θ)/cos(θ)
R² = dx² + dy²
