45° 대각 이동 — 삼각형 분해 대각거리를 X / Y 로 분해하는 원리
파라미터
50mm
45°
※ 45°일 때 X = Y
삼각비 계산 공식
// X, Y 분해
X=거리×cos(45°)
 =50.0×0.70711
 =35.355mm
Y=거리×sin(45°)
 =50.0×0.70711
 =35.355mm
유튜브 참고
삼각비 원리
cos(θ) — 가로 비율 빗변 대비 가로(X) 길이
X = 빗변 × cos(θ)
sin(θ) — 세로 비율 빗변 대비 세로(Y) 길이
Y = 빗변 × sin(θ)
tan(θ) — 기울기 가로(X) 대비 세로(Y) 비율
tan(θ) = Y / X
피타고라스 정리 직각삼각형 세 변의 길이 공식
빗변² = X² + Y²
📐 각도별 X, Y 비율 특징 • 30° : 가로(X)가 세로(Y)보다 긺
• 45° : 가로(X)와 세로(Y)가 똑같음
• 60° : 세로(Y)가 가로(X)보다 긺
