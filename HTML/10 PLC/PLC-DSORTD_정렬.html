DSORTD (32-bit Sort)

DSORTD(EN, s1, n, s2, d1, d2);

Sort BIN32 data from (s1) to (n) points in ASC/DESC order.

Operand Specifications
Operand Description
(s1) 테이블 선두 디바이스 (D100)
(n) 정렬 데이터 수 (2~10)
(s2) 1회 비교 데이터 수
(d1) 정렬 완료 비트 (M0)
(d2) 시스템 전용 디바이스 (D2000)
ST Program Example
// D100부터 n개 데이터 정렬
// s2=2 (1스캔당 2개씩 비교)
DSORTD(SM400, D100, 6, 2, M0, D2000);

"32비트 데이터는 항상 2워드(D100+D101)를 한 쌍으로 처리합니다."
