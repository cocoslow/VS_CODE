MTR (Matrix Input)

MTR(s, n, d, n2);

Reads 8*n switches into d starting from input s.

Operand Specifications
Operand Description
(s) 입력 선두 번호 (X0)
(n) 매트릭스 열 수 (8)
(d) 출력 선두 번호 (Y0)
(n2) 결과 저장 선두 (M0)
ST Program Example
// 8x8 키패드 읽기
// X0~X7 입력, Y0~Y7 출력
// M0~M63 에 결과 저장
MTR(X0, 8, Y0, M0);

"MTR은 스캔 방식이므로 속도가 중요할 경우 스캔 타임을 고려해야 합니다."
