RAMPQ(P) 경사 신호

초기값(s1)에서 최종값(s2)까지 n번에 걸쳐 부드럽게 값을 변화시킴

RAMPQ(
X0 EN
,
D0 초기(s1)
,
D2 최종(s2)
,
D10 결과(d1)
,
300 횟수(n)
,
M0 완료(d2)
);

경사 신호 시각화

TIME CALC
Target Reach Time
초 동안 상승
X
Scan Time 10ms (고정)
=
Set n to: 300

Device Monitoring

출력값 [D10] 0
현재 스캔 [(d1)+1] 0
완료 비트 [M0] OFF

🦾 시간 제어 핵심 정리

조드님, 이 명령어는 "시간"이 아니라 "스캔 횟수"로 동작해요!

1. 10ms 스캔 기준으로 계산하세요.
2. 1초를 원하면 n = 100
3. 3초를 원하면 n = 300
4. 10초를 원하면 n = 1000

이렇게 넣으면 기계가 부드럽게 가속합니다.

GX Works3 ST Example RAMPQ 
// 3초 동안 0에서 1000까지 증가 (스캔 10ms 기준)
RAMPQ(X0, 0, 1000, D10, 300, M0);
