ROTC(P) 로터리 테이블 제어

목표 위치까지 근거리(가장 빠른 방향)를 자동 판정하여 제어

ROTC(
1 EN
,
D10 테이블정보(s)
,
10 분할수(n1)
,
2 저속구간(n2)
,
M10 출력신호(d)
);

물리적 구조 이해 (확대 모드)

s+1 창구 번호 (목적지)
s+2 물품 번호 (이름표)

제어 신호 상태 D10 현재: 0

회전 방향 판정 (근거리) 정지
고속 정회전
정지 상태
고속 역회전
0점 (Home)
GX Works3 ST Example ROTC Control 
// s+1: 목적지 창구 번호
D11 := 0; 

// s+2: 옮길 물품 번호
D12 := 5; 

// 자동 근거리 제어 실행
ROTC(1, D10, 10, 2, M10);
