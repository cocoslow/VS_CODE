PWM (Pulse Modulation)

PWM(EN, s1, s2, d);

Generates a pulse with ON time (s1) and Period (s2) to device (d).

Operand Specifications
Operand Description
(s1) ON 시간 (1~65535 ms)
(s2) 주기 시간 (1~65535 ms)
(d) 펄스 출력 비트 (Y0)
ST Program Example
// 0.5초 ON, 1초 주기
// s1 < s2 조건을 만족해야 함
PWM(SM400, 500, 1000, Y0);

"PWM은 인버터 속도 제어나 히터 전력 제어에 주로 사용되는 핵심 명령어입니다."
