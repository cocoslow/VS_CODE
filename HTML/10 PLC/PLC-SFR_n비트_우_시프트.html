SFR(P) n비트 오른쪽 시프트

데이터(d)를 오른쪽으로 n비트 이동시키고, 마지막 비트를 SM700에 저장

SFR(
1 EN
,
D10 대상(d)
,
6 비트수(n)
);

D10 비트 데이터 동적 추적

Original Data [D10] (Click Bits)
⬇️
Shifted Result [D10] (Expected)
Decimal Value
0
Shift Type
Right Shift (0-Fill)

마지막 시프트 비트 (SM700)

0

오른쪽으로 밀려 나간 비트 중
가장 마지막 비트가 저장됩니다.

GX Works3 ST Example SFR(P) 
// D10을 오른쪽으로 6비트 시프트
D10 := 0;

// 시프트 실행 (Rising Edge)
SFR(1, D10, 6);

// D10 값과 SM700 상태 확인
