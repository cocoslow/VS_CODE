ROR(P) n비트 오른쪽 로테이션

데이터(d)를 오른쪽으로 n비트 회전시키고, 마지막 비트를 SM700에 저장

ROR(
1 EN
,
1 개수(n)
,
D10 대상(d)
);

D10 비트 로테이션 동적 추적

Original Data [D10] (Click Bits)
⬇️
Rotated Result (Expected)
Decimal Value
0
Rotate Type
Circular Right Rotate

마지막 로테이션 비트 (SM700)

0

회전되어 넘어간 비트 중
가장 마지막 비트가 저장됩니다.

GX Works3 ST Example ROR(P) 
// D10을 오른쪽으로 1비트 회전
D10 := 0;

// 로테이션 실행
RORP(1, 1, D10);

// 결과 확인:
// b0 데이터가 b15로 이동됨
// 마지막 이동 비트 -> SM700
