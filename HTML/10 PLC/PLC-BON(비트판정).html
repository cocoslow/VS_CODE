BON(P) 비트 판정

지정한 데이터(s)의 n번째 비트가 ON인지 체크하여 결과(d)에 저장

BON(
1 EN
,
D10 대상(s)
,
5 위치(n)
,
M0 결과(d)
);

D10 데이터 비트 모니터링

Decimal Value
0
Target Status
D10.5 = OFF

판정 결과 (M0)

OFF
M0 = 0
GX Works3 ST Example BON(P) 
// D10의 5번째 비트 확인
D10 := 0;

// 비트 ON 판정 실행
BON(1, D10, 5, M0);
