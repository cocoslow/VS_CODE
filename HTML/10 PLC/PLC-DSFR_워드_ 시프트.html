DSFR 동작 메커니즘

  • ●데이터가 낮은 주소 방향(D10 쪽)으로 1워드 통째로 밀립니다.
  • ●가장 끝(D10)에 있던 데이터는 밖으로 소멸됩니다.
  • ●데이터가 빠져나간 최상위 주소 빈자리는 0으로 채워집니다.
GX Works3 ST Example DSFR 
// DSFR(EN, n, d) 순서 적용
// D10부터 7개 워드 시프트

DSFRP(1, 7, D10);

// [결과 분석]
// D11 ➔ D10 (이동)
// D16 ➔ D15 (이동)
// 0   ➔ D16 (채움)
