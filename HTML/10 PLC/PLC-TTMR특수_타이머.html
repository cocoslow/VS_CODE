TTMR(P) 특수 타이머

입력이 ON인 시간을 측정하여 승수를 곱한 뒤 데이터로 저장

TTMR(
1 EN
,
0 0..2
승수(s)
,
D10 결과(d)
);

티칭 타이머 테스트

승수 설정 (Range: 0..2)
0.0s Ready
Raw Count (100ms unit)
0
Current Multiplier
x 1

최종 측정 결과 (D10)

0

버튼을 누르고 있는 동안 실시간으로 계산됩니다.

⚠ D11은 시스템용으로 자동 점유됩니다.
다른 데이터와 겹치지 않게 주의하세요!
GX Works3 ST Example TTMR(P) 
// s: 승수(0:x1, 1:x10, 2:x100)
// d: 결과 저장(D10, D11 점유)

D0 := 0; // 승수 설정 (0~2)

// X0가 ON인 동안 시간 측정
TTMR(X0, D0, D10);

// 결과 확인: D10 = 측정값 * 승수
